## ProEng

![GPL-3.0](https://img.shields.io/badge/LICENSE-GPL--3.0-red?style=for-the-badge&labelColor=0d0d0d)
![Python](https://img.shields.io/badge/PYTHON-3.x-FFD43B?style=for-the-badge&logo=python&logoColor=000&labelColor=0d0d0d)
![PyQt5](https://img.shields.io/badge/PYQT5-41CD52?style=for-the-badge&logo=qt&logoColor=fff&labelColor=0d0d0d)
![Linux](https://img.shields.io/badge/LINUX-FCC624?style=for-the-badge&logo=linux&logoColor=000&labelColor=0d0d0d)
![Status](https://img.shields.io/badge/STATUS-ATIVO-brightgreen?style=for-the-badge&labelColor=0d0d0d)

> suite desktop open source para engenharia de processos industriais.
> construída do zero para preencher uma lacuna real de ferramentas livres
> no contexto de modelagem, gestão e análise industrial em português.

```
STACK     : Python 3 · PyQt5 · GPL-3.0
PLATAFORMA: Linux · Windows · macOS
MÓDULOS   : 11 módulos integrados
TEMAS     : 14 temas dinâmicos
STATUS    : ativo · contribuições abertas
REPO      : github.com/sasal0519/proeng
```

| MÓDULO | DESCRIÇÃO | TIPO |
|--------|-----------|------|
| BPMN Modeler | modelagem de processos com swimlanes | diagrama |
| Flowsheet / P&ID | diagramas de processo industrial | diagrama |
| Gantt | cronogramas com dependências | gestão |
| Kanban / SCRUM | gestão ágil de tarefas | gestão |
| Ishikawa | análise de causa raiz | análise |
| 5W2H | resolução estruturada de problemas | análise |
| PDCA | ciclos de melhoria contínua | análise |
| Script Engine | Python embutido para automação | core |

```diff
+ interface construída com PyQt5 · design neo-brutalista
+ 14 temas dinâmicos com troca em tempo real
+ Script Engine: execute Python dentro da própria suite
+ exportação de diagramas e relatórios
- sem telemetria. sem assinatura. sem nuvem obrigatória.
```
