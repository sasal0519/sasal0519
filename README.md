> suite desktop open source para engenharia de processos industriais.
> construída do zero para preencher uma lacuna real de ferramentas livres
> no contexto de modelagem, gestão e análise industrial em português.

```
+---------------------+------------------------------------------+----------+
| MODULO              | DESCRICAO                                | TIPO     |
+---------------------+------------------------------------------+----------+
| BPMN Modeler        | modelagem de processos com swimlanes     | diagrama |
| Flowsheet / P&ID    | diagramas de processo industrial         | diagrama |
| Gantt               | cronogramas com dependencias             | gestao   |
| Kanban / SCRUM      | gestao agil de tarefas                   | gestao   |
| Ishikawa            | analise de causa raiz                    | analise  |
| 5W2H                | resolucao estruturada de problemas       | analise  |
| PDCA                | ciclos de melhoria continua              | analise  |
| Script Engine       | Python embutido para automacao           | core     |
+---------------------+------------------------------------------+----------+
```
